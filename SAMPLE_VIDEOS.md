### How to Use
1. Copy a YouTube video link from this file
2. Run the analyzer: `python futsal_analyzer.py`
3. Paste the link when prompted
4. Optionally enter a start time (MM:SS format)
5. Calibrate the field with 6 points
6. Analysis will begin

## Tips
- **Best Results:** Videos with fixed side camera view (typical futsal broadcast angle)
- **Video Quality:** Supports up to 1080p (automatically downgrades if unavailable)
- **Duration:** Works with any video length
- **Live Streams:** Works with active YouTube live streams too
