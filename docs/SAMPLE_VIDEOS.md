### How to Use

1. Copy a YouTube video link from this file.
2. Run the analyzer: `python -m futsal_analytics` (or `futsal-analytics`).
3. Paste the link when prompted.
4. Optionally enter a start time in `MM:SS` format.
5. Calibrate the field with the 6 control points.
6. Analysis begins automatically.

## Tips

- **Best results**: Fixed side-camera view — the standard futsal broadcast angle.
- **Video quality**: Supports up to 1080p (automatically falls back if unavailable).
- **Live streams**: Works with active YouTube live streams.
